# Week 13: Quiz 3 – Advanced Embedded Systems Programming

This week is dedicated to **Quiz 3**, the final assessment covering Phase 3: Advanced Techniques.

## Quiz Structure
The assessment is divided into two parts:

1. **Theory (Paper-based)**:
   - Program flow control and optimization.
   - I/O manipulation and bitwise logic.
   - Advanced techniques and state machines.
2. **Programming (Practical)**:
   - Implementing complex multi-mode controllers.
   - Optimizing code for speed and memory.
   - Handling asynchronous events and real-time timing.
