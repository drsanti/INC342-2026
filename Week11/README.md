# Week 11: Programming Optimization Techniques

Welcome to Week 11! This week, we focus on **Code Optimization Strategies** for embedded systems, focusing on execution speed and memory efficiency.

## Learning Objectives
- **Execution Speed**: Reducing function overhead, using the `register` keyword, and bit shifting.
- **Memory Efficiency**: Ternary operators, logic consolidation, and eliminating intermediate variables.
- **Best Practices**: Naming conventions and balancing readability with conciseness.
