## Quiz 1 scope (from course outline)

* **Paper-based (handwritten):** Embedded system concepts, development environment (CLI, workflow, testing commands), firmware development (workflow, build process, project structure), Embedded C fundamentals, event-driven programming (events, callbacks, timers/intervals, state machines, ADC basics).
* **Programming:** Embedded C syntax, `main`, `#include`, `serial_printf`; project creation, compilation, flashing, monitoring; event-driven applications (LED blinking, switch control, callbacks, event state filtering).
