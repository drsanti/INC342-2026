# Week 09: Program Flow Control and I/O Manipulation

Welcome to Week 09! This week, we begin **Phase 3: Advanced Techniques**, focusing on how to control program flow and manipulate inputs/outputs efficiently in embedded systems.

## Learning Objectives
By the end of this week, we will explore:
- **Program Flow Control**: Using conditional logic, timing diagrams, and bitwise optimization.
- **I/O Manipulation**: Understanding the Sensor-Controller-Actuator model and handling Active High/Low configurations.
- **Bitwise Logic**: Using masks (`&`, `|`) and shifts to control hardware pins directly.
- **Binary Representation**: Displaying data using 4-bit binary patterns on LEDs.
